### Contibute

1.  **Clone the website**

    ```shell
    git clone https://github.com/SatvikVirmani/we-covid-track/
    cd we-covid-track
    npm install
    ```
    
2.  **Start development**

    ```shell
    gatsby develop
    ```

3. **Make a pull request**

To publish changes just push to repository
